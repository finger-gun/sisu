# @sisu-ai/skill-test-gen

Test generation skill for Vitest with coverage guidance.

## Install

```bash
pnpm add @sisu-ai/skill-test-gen
```

## Usage

```bash
mkdir -p .sisu/skills/test-gen
cp -R node_modules/@sisu-ai/skill-test-gen/* .sisu/skills/test-gen/
```
