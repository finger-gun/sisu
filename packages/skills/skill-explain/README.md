# @sisu-ai/skill-explain

Explanation skill for code and architecture walkthroughs.

## Install

```bash
pnpm add @sisu-ai/skill-explain
```

## Usage

```bash
mkdir -p .sisu/skills/explain
cp -R node_modules/@sisu-ai/skill-explain/* .sisu/skills/explain/
```
