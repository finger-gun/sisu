# @sisu-ai/vector-core

Provider-agnostic vector types for Sisu. Keep tools and middleware pluggable and portable.

Exports:
- `Embedding`
- `VectorRecord`
- `QueryRequest`
- `QueryResult`

Use with a vector adapter like `@sisu-ai/tool-vec-chroma`.
